Question-1:What are the different ways to select an element in the Dom?

Use getElementById if selecting one unique element. Use getElementsByClassName or querySelectorAll for multiple elements. Use querySelector for flexibility with CSS selector

Question-2:What is the different between innerHTML,innerText and textContent?

Use innerHTML if i need HTML formatting. Use innerText if i want only visible text. Use textContent if i want all text, even hidden ones.

Question-3:What is event delegration in the Dom?

When handling many child elements. When working with dynamic content (new elements added later).

Question-4:What is event bubbling in the Dom?

When i need events to reach parent elements. When using Event Delegation.
